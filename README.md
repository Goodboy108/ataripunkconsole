# ataripunkconsole
This is a audio synth based on a NE556 IC dual timer. it uses 2 potentiometers to control the frequency and oscillation of the sound.
This project is made to recreate the tones atari retro games made and by rotating 2 knobs, you can alter the sound to how you want it!
motivation - I made this project because it is comparatively easier to make then other projects because this is my first ever hardware project. I also love the game console theme of it and also how it is not expensive to produce - what i thought. cause the pcb in my country is like 30$ gng, im cooked.
